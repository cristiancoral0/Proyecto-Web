Introducción

El proyecto tiene como fin generar el desarrollo de una aplicación web que interactúe de manera dinámica con el usuario que la manipula, se tiene pensado que la aplicación sea el juego clásico de culebrita que traían los antiguos celulares.

Funcionalidad

La idea del desarrollo de la aplicación consiste inicialmente en la ejecución de un login que lo contraste con una base de datos en SQL para acceder a su perfil el cual contara con puntajes y estadísticas de los juegos ejecutados.

Desarrollo

El proyecto será ejecutado en código HTML5 y CSS3, con bootstrap para mejorar la estética de la interfaz, se tiene previsto como avance culminar la parte del login y desarrollar las interfaces de mapas o escenarios de los niveles.

CONTEXTO

Para entrar en materia con el desarrollo o contexto de la ejecución de la aplicación, inicialmente basaremos la ejecución de la publicación en el diagrama siguiente, en él se puede identificar que la aplicación estará montada en un servidor Web, con intermedio de la herramienta que maneje la funcionalidad de GIT, para ello usaremos GitHub y Azure para con sistema operativo Ubuntu. El diagrama también muestra la función del desarrollador web quien en este escenario también será el DBA, y la funcionalidad de cliente que describe que la aplicación funcionara tanto para equipos de cómputo como para dispositivos móviles con sistema Android o IOS.

Primera Etapa

Para esta primera etapa se creara un repositorio con GitHub con el fin de almacenar el código y documentar cada transformación que tenga el mismo.

Segunda Etapa

Para la segunda etapa se pretende ejecutar el código beta en la web a través de un servidor Ubuntu en Azure con los respectivos ajustes de sesión.

Tercera Etapa

Como última etapa para la versión beta se pretende pulir novedades y ejecutar pruebas en búsqueda de fallas para su corrección y salida al público
